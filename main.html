Making the Convolution Biologically Plausible for SNN Image Recognition¶

We typically implement convolutions as a multiplication between a kernel and the subsets of the image it can project to.

In this notebook we try to show that the non-memoryless-ness of the LIF model can be leveraged to represent a pseudo-convolution operation, done entirely in the additive spiking domain, and without calling upon any (biologically implausible) multiplication.

We think this could be especially useful to translate the idea of convolutional layers (which perform better for feature detection and hence image recognition) into biologically sound neural networks models.

In this case, the left layer represents different sensory neurons and their 'weights' (spike interval), and the kernel in the middle represents any additional effect caused by non-sensory feedback neurons/dendrites to their respective sensory neurons.¶

Unlike typical convolutional neural networks, we don't have a kernel which just "travels" across the image. Instead, there are many stationary kernel across the same image. By stimulating the dendrites/feedback neurons in interesting ways, we are able to change the kernel and extract features from the image.

In a standard CNN, each kernel has to be trained into a formation: here instead, the brain learns the spatio-temporal stimulation 'functions' applied to the feedback neurons, and is then able to repeat that learned formation to extract desired features.

We seek to define a 'convolution' operation in the spiking domain -with an output as the later layer neurons-, without using the traditional dot product method shown in the picture, but only with addition in leaky neuron models¶

from scipy.special import softmax

class N():
        def __init__(self, input_current, params='default'):
            if params == 'default':
                self.physical_params = lif_model.default_pars(T=150)
            else:
                self_physical_params = params

            self.input_current = input_current
            self.voltage, self.output_spikes = lif_model.run_LIF(physical_params, Iinj=input_current, stop=False)
            self.avg_out = len(self.output_spikes)/150

        def plot(self):
            lif_model.plot_volt_trace(self.physical_params, self.voltage, self.output_spikes)

        def __float__(self):
            return self.avg_out
        
class N_Container():   
    def __init__(self, size, frequency_array):
        f = {0: 0, 50: 300, 100 : 400} #frequency dictionary- turns average spikes into current
        assert isinstance(size, tuple)
        assert isinstance(frequency_array, np.ndarray)
        
        dim = size[0], size[1]
        assert (dim == frequency_array.shape)
        self.arr = np.array([N(f[freq]) for freq in frequency_array.flatten()]).reshape(dim)
    
    def __iter__(self):
        return self.arr
    
    def __getitem__(self, index):
        return self.arr[index]
    
    def get(self):
        output = np.zeros_like(self.arr)
        for iii, line in enumerate(self.arr):
            for jjj, neuron in enumerate(line):
                output[iii][jjj] = float(neuron)
        arr = np.floor(output*400)
        return np.array(arr, dtype=np.float64)
    
    def getsoftmax(self):
        return np.floor(self.get())
    
    def __len__(self):
        return len(self.arr)
